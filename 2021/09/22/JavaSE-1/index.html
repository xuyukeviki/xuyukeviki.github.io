static修饰词

静态属性：static开头的属性

静态方法块：static{}包裹的代码块

普通属性：非static定义的属性

普通方法块：{}包裹的代码块

构造函数：类名相同的方法

方法：普通方法

public class LifeCycle {
     //静态属性
     private static String staticField = getStaticField();
     //静态方法块
     static {
     System.out.println(staticField);
     System.out.println("静态方法初始化");
     }
     //普通属性
     private String field = getField();
     //普通方法块
     {
     System.out.println(field);
     }
     //构造函数
     public LifeCycle() {
         System.out.println("构造函数初始化");
     }
     public static String getStaticField() {
         String statiFiled = "Static Field Initial";
         return statiFiled;
     }
     public static String getField() {
         String filed = "Field Initial";
         return filed;
     }
     //主函数
     public static void main(String[] argc) {
     new LifeCycle();
     }
}

这段代码的执行结果就反应了它的初始化顺序

静态属性初始化 静态方法块初始化 普通属性初始化 普通方法块初始化 构造函数初始化

package com.baidu.www.oop.demo07;

public class Student {
    private static int a;//静态变量
    private double b;//非静态变量
    public static void run(){
        System.out.println("runrunrun!");
    }
    public void go(){
        System.out.println("gogogo!");
    }
    public static void main(String[] args) {
        Student student = new Student();
        System.out.println(Student.a);
        //System.out.println(Student.b);
        //因为非静态变量，必须实例化之后才能访问
        System.out.println(student.a);
        System.out.println(student.b);
        new Student().go();//非静态方法必须实例化之后再使用
        Student.run();//静态方法可以直接调用
        run();
    }
}

package com.baidu.www.oop.demo07;
//静态导入包
import static java.lang.Math.random;
public class Teacher {
    //2
    {
        System.out.println("调用匿名代码块");
    }
    //1而且只调用一次
    static {
        System.out.println("调用静态代码块");
    }
    //3
    public Teacher() {
        System.out.println("使用构造器");
    }
    public static void main(String[] args) {
        Teacher teacher2;//会输出调用静态代码块
        System.out.println("--------------------------------------------");
        Teacher teacher = new Teacher();//会输出带哦用匿名代码块以及使用构造器
        System.out.println("============================================");
        Teacher teacher1 = new Teacher();//会输出带哦用匿名代码块以及使用构造器
        System.out.println(random());//可以直接使用静态导入的东西
    }
}
/*
代码块
1.代码块的作用：用来初始化类
2.代码块如果有修饰的话，只能使用static修饰词
3.分类：静态代码块vs非静态代码块
4.静态代码块
	内部可以有输出语句
	随着类的加载而执行
	只在加载类的时候执行一次
	作用：初始化当前类的属性
	如果一个类中定义了多个静态代码块，则按照声明的先后顺序执行
5.非静态代码块
	内部可以有输出语句
	随着类的实例化而执行(对象的创建)
	每创建一个对象，就执行一次非静态代码块
	作用：可以在创建对象时，对对象的属性等进行初始化
	
	
	
对象属性可以赋值的位置：
1.默认初始化
2.显式初始化
3.构造器中初始化
4.对象.属性赋值
5.在代码块中赋值


静态之中不可以调用非静态(除非在静态中创建一个对象)，非静态之中可以调用静态，因为静态产生于非静态之前
*/

标识符的使用

命名规则：

**1.**由26个英文字母大小写0-9，__，$组成

**2.**数字不可以开头

**3.**不可以使用关键字或者保留字

**4.**严格区分大小写，长度无限制，不能包含空格

命名规范：

包名：多单词组成时所有字符都小写：xxxyyyzzz

类名，接口名：多单词组成时，所有单词的首字母大写：XxxYyyZzz

变量名，方法名：多单词组成时，第一个单词首字母小写，第二个单词开始每个单词首字母大写：xxxYyyZzz

常量名：所有字母都大写。多单词时每个单词用下划线连接：XXX_YYY_ZZZ
