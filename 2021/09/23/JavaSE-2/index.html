JavaSE基础阶段的总结(二)

变量

java的变量必须先声明，后使用

##变量的作用域：

类变量,实例变量，局部变量

public class Variable{
    static int allClicks = 0;//类变量
    String str = "hello world";//实例变量
    public void method(){
        int i = 0;//局部变量
    }
}
/*
  类变量和实例变量的区别在于有没有这个static修饰词
  局部变量是定义在类的方法中的
*/



public class Demo1 {
    //常量定义 final修饰词，与static不区分先后顺序，用大写字母来表示
    static  final  double PI = 3.14;
    //类变量
    static  double Salary = 1;
    //属性：变量

    //实例变量：从属于对象，如果不自行初始化，这个类型的默认值是0，0.0
    //布尔值：默认值都是false
    //除了基本类型，其余的默认值都是null
    int age;
    String name;
    //main方法
    public static void main(String[] args) {
        //局部变量；必须声明和初始化值
        int i=10;
        System.out.println(i);

        //变量类型 变量名字= new Demo1；
        Demo1 demo1 = new Demo1();
        System.out.println(demo1.name);
        System.out.println(demo1.age);

        //类变量 static
        System.out.println(Salary);
        System.out.println(PI);
    }
}

类型转换

基本数据类型之间的运算规则：

（讨论不包含boolean类型的七种基本数据类型的运算，原因，java中的true or false和c语言不一样不能直接代表1 or 0）

1.自动类型提升

byte,char,short-->int-->long-->float-->double

当容量小的数据类型的变量雨容量大的数据类型的变量做运算时，结果自动提升为容量大的数据类型；当byte，char，short三者做运算，会自动转化为int类型、

强制类型转换

b = (int)a

注：可能发生精度损失

例外情况

long L1=2132131231321312//会报错，因为没加上L会默认按照4bytes来存储
long L1=2132131231321312L//不会报错，因为加上L会按照8bytes来存储

string引用数据类型

应用数据类型的默认值为null

注意点：string只能和其他8种数据类型（包括boolean）做运算，且只能是连接运算且结果为String

class StringTest{
	public static void main(String[] args){
		String s1="hello world!";//声明String类型的变量
		boolean b=true;
		int c=2；
		char d='a';
		System.out.println(s1+b);//会输出hello world！true
		System.out.println(s1+c);//会输出hello world！2

不同进制表示方式

/* 二进制（binary）：0，1，满2进1.以0b或0B开头
   十进制（decimal）：0-9，满10进1
   八进制（octal）：0-7，满8进1，以数字0开头表示
   十六进制（hex）：0-9及A-F，满16进1，以0x或0X开头表示*/
class BinaryTest{
    public static void main（String[] args）{
        int num1=0b11011;//二进制
        int num2=110;//十进制
        int num3=0127;//八进制
        int num4=0x110AD;//十六进制
    }
}

java运算符

/:java中的除法运算符本质上来说是整除运算符，但是，若是其中一个数为浮点数，就会得到浮点数的结果

++、–：自加自减运算符（与c语言相同，a++和++a）

若a是10，则++a=11；a++=10

&和&&的区别：&当符号左边是false的时候会继续执行右边的运算；&&当符号左边是false的时候就不会执行右边的运算。

|和||的区别：|当符号左边是false的时候会继续执行右边的运算；||当符号左边是false的时候就不会执行右边的运算。

<<：对应的二进制数左移两位，并在末尾补充两个0。（左移一位原十进制乘2）
21<<2=21*2*2=84
>>：对应的二进制数右移两位，并在开头补充两个，若最高位为1则补1，是0则补0。（右移一位原十进制/2）
>>>:无符号右移，就是不论最高位，都补上0
&：二进制位进行&运算，只有1&1结果才是1，否则就是0
|：二进制进行|运算，只有0|0时才会是0
^：相同二进制位进行^运算，结果是0；不相同二进制位^运算结果是1。1^0是1，1^1是0，0^0是0，0^1是1
~：各二进制码按补码各位取反
